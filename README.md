Talksera is a full-stack English learning platform targeting Arabic-speaking learners. The frontend is a modern
Single Page Application (SPA) built with React 18 + TypeScript, consuming the NestJS REST API and
WebSocket gateway. It delivers a rich, interactive learning experience that mirrors the design defined in the
Talksera UI prototype.
The interface supports all CEFR levels (A1–C2), a freemium subscription model via Stripe, AI-powered real-time
chat via Socket.io, audio recording for pronunciation, and a full gamification system with XP, streaks, and badges
— all managed via a centralized Redux Toolkit store with RTK Query for server state.
